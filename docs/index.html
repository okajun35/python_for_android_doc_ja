python-for-android¶

python-for-androidはPythonコードをスタンドアローンのアンドロイドAPKにパッケージ化して他のAndroidアプリと同じようにPlayストアなどのマーケットプレイスに渡したり、インストールしたりアップロードしたりできるオープンソースのビルドツールです。このツールはもともとはクロスプラットフォームなグラフィカルフレームワークである Kivy<http://kivy.org/#home> ように開発されましたが、現在は複数のブートストラップをサポートしており、Android用の他の種類のPythonアプリを簡単にパッケージ化できます。

python-for-androidは、2つの主要な操作をサポートしています。まず、Pythonインタプリタ、その依存関係、バックエンドライブラリ、AndroidデバイスのPythonコードをコンパイルできます。この段階は完全にカスタマイズ可能であり、好きなだけ多くのコンポーネントをインストールできます。その結果、別々の名前、アイコン、Pythonコードなどを使用していても、任意の数の異なるAPKを生成するために使用できるスタンドアロンのAndroidプロジェクトです。python-for-androidの2番目の機能は、これらのディストリビューションへの簡単なインターフェイスを提供することです。そのようなプロジェクトからビルドパラメータとPythonコードを使ってPython APKを生成します
